Accordion

Accordions are simple content blocks that expands or contracts on toggle. They can be used to hide extra optional information, or to build a step wise explaination.

To build an accordion, create a button with accordion class, and immediately create a <div> with -panel class.

For example

<button class="accordion">toggle</button>
<div class="-panel"> Content </div>
Content

Note that you need to include beauter.min.js at the end for the dynamic event triggering. See how here.

You can create nice interactive interfaces using accordions.
